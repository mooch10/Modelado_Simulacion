🔧 MÓDULOS PRINCIPALES (2 ARCHIVOS)
═══════════════════════════════════════════════════════════════════════════════

🔹 penales_montecarlo.py (250+ líneas) ⚙️ MOTOR DE SIMULACIÓN
   └─ Contenido:
      • Clase JugadorPenal (dataclass)
        └─ Atributos: nombre, peso, altura, precisión, velocidad, experiencia
      
      • Clase PorteroPenal (dataclass)
        └─ Atributos: nombre, peso, altura, reflejos, experiencia
      
      • Clase SimuladorPenales
        ├─ __init__(semilla=None)
        ├─ calcular_probabilidad_gol(jugador, portero) → float [0-1]
        ├─ simular_penal(jugador, portero) → bool (gol/no gol)
        ├─ simular_tanda(jugadores, portero, n_simulaciones) → dict
        │  └─ Retorna: {
        │       'goles_totales_esperado': float,
        │       'goles_std': float,
        │       'ic_lower': float,
        │       'ic_upper': float,
        │       'probabilidad_ganar': float,
        │       'probabilidad_empatar': float,
        │       'probabilidad_perder': float,
        │       'prob_por_jugador': {nombre: prob, ...}
        │     }
        └─ FÓRMULA: P(gol) = w1*precisión + w2*velocidad + ...
   
   Independencias:
   • NumPy (cálculos)
   • dataclasses (estructuras)
   
   📌 PARA QUIÉN: Desarrolladores, usuarios Python


🔹 utilidades_penales.py (400+ líneas) 📊 BASE DE DATOS + HERRAMIENTAS
   └─ Contenido:
      • JUGADORES_CONOCIDOS: Dict[str, tuple]
        └─ 54 jugadores reales con: peso, altura, precisión, velocidad, 
           experiencia, país, emoji
        └─ Países: Argentina, Portugal, Brasil, Alemania, Francia, España, 
           Italia, etc.
      
      • PORTEROS_CONOCIDOS: Dict[str, tuple]
        └─ 20 porteros reales con: peso, altura, reflejos, experiencia, 
           país, emoji
      
      • Clase GeneradorDatos (métodos estáticos)
        ├─ obtener_jugador_conocido(nombre) → JugadorPenal
        ├─ obtener_portero_conocido(nombre) → PorteroPenal
        ├─ obtener_jugadores_aleatorios(cantidad=11, por_equipo=True) 
        │  └─ ⭐ NUEVO: Agrupa por país si por_equipo=True
        ├─ obtener_portero_aleatorio() → PorteroPenal
        ├─ listar_jugadores() → List[str]
        ├─ listar_porteros() → List[str]
        └─ obtener_jugadores_por_pais() → Dict[str, List[str]]
      
      • Clase ExportadorResultados (herramienta auxiliar)
        ├─ exportar_json(resultados, archivo) → json file
        ├─ exportar_csv(resultados, archivo) → csv file
        └─ exportar_html(resultados, archivo) → html file
   
   📌 PARA QUIÉN: Desarrolladores, usuarios avanzados
